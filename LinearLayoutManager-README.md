# LinearLayoutManagerDivider
 一个通用简易的RecyclerView垂直布局管理器的分割线
 
 ## Method and Attribute

常用方法
```
// LayoutManager - > 水平方向
LinearLayoutManagerDivider.getHorizontalDivider()
// LayoutManager - > 垂直方向
getVerticalDivider()
```

## About


注意：
1. 使用项目提供的适配器时，头、脚、状态布局不会加分割线
2. 最后一个item（即 脚布局上面的item）不会加分割线
